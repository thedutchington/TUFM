# Start Guide

1. Install **Forge 47.4.10** for Minecraft 1.20.1 at the link:
   https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html

2. Launch Minecraft once with Forge selected, then close the game.

3. Copy all files from the `Mods` folder into your `.minecraft/mods` directory.

4. Launch Minecraft using the Forge 47.4.10 profile and enjoy.
